# 24時間シート エクスポート機能 ユーザーガイド

## 📋 概要
24時間シート（ケアプラン）を3つの形式でエクスポートできます。家族への配布、記録保管、データ分析などに活用できます。

---

## 🎯 使い方

### 方法1: UIからエクスポート（簡単）

1. **アプリにアクセス**
   - https://3000-ijob28wtyr70pn2awiqwk-02b9cc79.sandbox.novita.ai/

2. **左側の24時間シート（PCの場合）を確認**
   - 入居者様の情報が表示されている

3. **エクスポートボタンをクリック**
   - 「🖨️ 印刷」ボタン: 印刷用HTMLページを新しいタブで開く
   - 「📄 CSV」ボタン: CSVファイルをダウンロード

---

### 方法2: API経由でエクスポート（開発者向け）

**エンドポイント:**
```
GET /api/residents/{resident_id}/care-plans/export?format={format}
```

**パラメータ:**
- `resident_id`: 入居者ID（例: 1）
- `format`: 出力形式
  - `html`: 印刷用HTML
  - `csv`: CSV形式
  - `json`: JSON形式

**例:**
```bash
# HTML形式
curl "http://localhost:3000/api/residents/1/care-plans/export?format=html"

# CSV形式
curl "http://localhost:3000/api/residents/1/care-plans/export?format=csv" -o 24時間シート.csv

# JSON形式
curl "http://localhost:3000/api/residents/1/care-plans/export?format=json" -o 24時間シート.json
```

## 📄 出力形式の詳細

### 1. HTML形式（印刷用）

**用途:**
- 家族への配布
- 紙媒体での保管
- 施設内掲示

**特徴:**
- ✅ A4サイズに最適化
- ✅ 印刷ボタン付き
- ✅ ヘッダーに入居者情報と日付
- ✅ テーブル形式で見やすい
- ✅ ステータス（フィット済み/計画）を色分け表示
- ✅ フッターに印刷日時

**印刷手順:**
1. 「🖨️ 印刷」ボタンをクリック
2. 新しいタブで印刷用ページが開く
3. ページ内の「🖨️ 印刷する」ボタンをクリック
4. プリンターを選択して印刷

**サンプル出力:**
```
┌─────────────────────────────────────────────┐
│ 24時間シート（生活リズムシート）              │
│ 入居者: 岡田 一輝 様  介護度: 要介護4        │

├────┼──────────┼──────────────┼──────────┤

│08:00│朝食      │椅子に深く...   │計画      │
│...  │...       │...             │...       │
└────┴──────────┴──────────────┴──────────┘
印刷日時: 2026-03-21 15:30:00
```

### 2. CSV形式

**用途:**
- Excelでの編集・加工
- データ分析
- 他システムへのインポート

**特徴:**
- ✅ Excel互換（UTF-8 BOM付き）
- ✅ カンマ区切り
- ✅ ヘッダー行付き
- ✅ ダブルクォートでエスケープ
- ✅ ファイル名に入居者名と日付

**ファイル名例:**
```
岡田 一輝_24時間シート_2026-03-21.csv
```

**サンプル出力:**
```csv
時刻,活動,詳細・留意点,状態
"07:00","起床・洗面","カーテンを開け日光を入れる。右側から声をかける。","フィット済み"
"08:00","朝食","椅子に深く座るよう促す。お茶は熱めを希望。","計画"
"10:00","排泄介助","トイレ誘導。立位保持を介助。","フィット済み"

```

**Excelでの開き方:**
1. CSVファイルをダウンロード
2. Excelで「データ」→「テキストから」→CSVファイルを選択
3. 文字コード「UTF-8」を選択
4. 区切り文字「カンマ」を選択

## 📊 活用シーン

### シーン1: 家族への配布
**形式**: HTML（印刷用）
1. HTMLエクスポート
2. 印刷してファイリング
3. 家族面談時に配布
4. 入居者様の1日の流れを説明

### シーン2: 記録保管
**形式**: CSV + JSON
1. 月末にCSVとJSONでエクスポート
2. クラウドストレージに保存
3. 年次監査時に提出
4. データの完全性を担保

### シーン3: データ分析
**形式**: CSV
1. CSVエクスポート
2. Excelで開く
3. フィット済み項目の割合を集計
4. Day14時点での完成度を評価

### シーン4: 施設内共有
**形式**: HTML（印刷用）
1. HTMLエクスポート
2. カラー印刷
3. リビングやスタッフルームに掲示
4. 全職員が参照できるようにする

### シーン5: 他施設への移管
**形式**: JSON
1. JSONエクスポート
2. メールまたはUSBで移管先施設へ送付
3. 移管先でインポート
4. ケア内容を継続

## 🔧 トラブルシューティング

### 問題1: CSVがExcelで文字化けする
**原因**: 文字コードの問題  
**解決方法**:
1. Excelで「データ」→「テキストから」でインポート
2. 文字コード「UTF-8」を明示的に選択

### 問題2: 印刷時にレイアウトが崩れる
**原因**: ブラウザの印刷設定  
**解決方法**:
1. 用紙サイズをA4に設定
2. 余白を「標準」または「狭い」に設定
3. 「背景のグラフィック」をオンにする

### 問題3: エクスポートボタンが表示されない
**原因**: モバイル表示になっている  
**解決方法**:
1. PCまたはタブレットでアクセス
2. ブラウザをフルスクリーンで開く
3. または、APIを直接叩く
