## 特別養護老人ホーム（特養）とは何か

### 🏥 医療施設ではなく「生活の場」

特養は**病院ではありません**。ここは**人生最後の「家」**です。

- **入居者**: 要介護3〜5の方が中心（重度の介護が必要）
- **平均入居期間**: 数年〜最期まで（看取りを含む）
- **目的**: 治療ではなく「その人らしい生活を支える」こと

### 🏠 ユニット型特養の特徴

現代の特養の主流は「ユニット型」：

```
1ユニット = 10名前後の入居者 + 専属スタッフ
├── 個室（プライベート空間）
├── リビング・ダイニング（共有空間）
└── キッチン（家庭的な雰囲気）

施設全体 = 複数ユニット（例：5ユニット = 50名規模）
```

**なぜユニット型？**
- 大規模施設の「施設感」を減らし、小規模な「家庭」を再現
- 馴染みのスタッフとの関係構築
- 一人ひとりの生活リズムに合わせたケア

### 👥 一日の主な登場人物

| 役割 | 人数（ユニットあたり） | 仕事内容 |
|------|----------------------|---------|
| **介護職員** | 日勤3-4名、夜勤1-2名 | 食事・入浴・排泄介助、生活支援 |
| **看護師** | 日勤1-2名 | 医療的ケア、健康管理 |
| **ケアマネジャー** | 施設全体で数名 | ケアプラン作成・調整 |
| **相談員** | 施設全体で数名 | 家族対応、入退所調整 |
| **リハビリ職** | 週数回訪問 | 機能訓練 |
| **栄養士** | 施設全体で数名 | 食事管理 |

## 入居者の方々の暮らし

### 👴 岡田一輝さん（78歳・要介護4）の例

```
07:00  起床 - 「あ、もう朝か。カーテン開けてくれる？」
       職員が右側から声をかけ、カーテンを開ける
       日光が入ると「気持ちいいね」と笑顔

08:00  朝食 - 和食が好き。味噌汁は熱めを希望
       刻み食だが「味はわかるよ」と完食
       食後のコーヒーが楽しみ「これがないと始まらない」

10:00  トイレ誘導 - 自力で立ち上がれないが、声かけで協力的
       「ありがとうね、いつもすまないね」

12:00  昼食 - 今日は肉じゃが「おふくろの味だ」と懐かしむ
       職員と昔の話で盛り上がる

14:00  午睡 - 静かな環境でぐっすり
       「昔は昼寝なんてしなかったけどね」

15:00  おやつ - 季節の和菓子と緑茶
       「桜餅か、もう春だねぇ」

17:00  中庭へ - 車椅子で桜を見に行く
       「きれいだねぇ。来年も見られるかな」
       職員「来年も一緒に見ましょう！」

18:00  夕食 - 好きなメニュー「天ぷらは若い頃よく食べたなぁ」

20:00  入浴 - 週3回、今日は入浴日
       「気持ちいいねぇ」とリラックス

21:00  就寝準備 - 照明を落とし、静かな音楽
       「じゃあ、おやすみ」

22:00  夜間巡視 - 2時間おきに様子確認
00:00  巡視 - 熟睡中、穏やかな表情
02:00  巡視 - トイレ誘導「起こしてごめんね」
04:00  巡視 - 熟睡中
```

### 🌟 これが「生活」

岡田さんの一日は、病院の「患者」ではなく、**一人の人間としての「暮らし」**です。

- コーヒーの好み
- 季節の移ろいを感じる心
- 家族への思い
- 職員への感謝の言葉
- 来年への希望

**これらすべてが「その人らしさ」であり、ケアの出発点です。**

## ケア記録は「人生最後の物語」

### 📝 記録は「法的義務」だけど、本質は違う

法律では「ケア記録の作成・保存」が義務付けられています。でも、それは**最低限のルール**。

**記録の本当の価値**は、**入居者の人生最後の日々を「物語」として残すこと**です。

### 📖 記録が紡ぐ物語の例

#### 2026年3月19日（水）

```
08:15 【起床】
岡田さん、いつもより15分遅い起床。
「よく寝たなぁ」と満足そう。
右側から声をかけると、すぐに反応。
カーテンを開けると「春の日差しだね」と笑顔。

→ スタッフの気づき: 最近は8:15まで熟睡されている。
   7:00起床は早すぎるかもしれない。
```

#### 2026年3月20日（木）

```
08:00 【朝食】
今朝もコーヒーをリクエスト。
「このコーヒーの味、娘が入れてくれたのに似てるんだよ」
と娘さんの話をされる。

→ スタッフの気づき: コーヒーが娘さんとの思い出につながっている。
   家族面会時に一緒にコーヒーを飲む時間を作れないか？
```

#### 2026年3月21日（金）

```
17:00 【中庭散歩】
車椅子で桜を見に行く。
「きれいだねぇ。来年も見られるかな」と呟かれる。
「来年も一緒に見ましょう！」と返すと、
「そうだね、楽しみにしてるよ」と笑顔。

→ スタッフの気づき: 桜が岡田さんにとって特別な意味を持っている。
   来年の春、また一緒に桜を見たい。
```

---

### 💭 これらの記録が意味するもの

この3日間の記録から見えてくるもの：

1. **睡眠リズムの変化** → 起床時刻を8:15に調整できる
2. **コーヒーと娘さんの思い出** → 家族との時間を大切にできる
3. **季節への感受性** → 桜の季節を一緒に楽しめる

そして、もし岡田さんが1年後に旅立たれたとき、ご家族に渡せる「最後の1年の物語」になります。

**「お父さん、施設でもコーヒー楽しんでたんですね」**  
**「桜、毎年楽しみにしてたんですね」**

これが**記録の本質**です。

## 法律・制度の側面（でも本質じゃない）

### 📜 法的に必要な記録の例

介護保険法や厚生労働省の基準により、以下の記録が義務付けられています：

| 記録種別 | 頻度 | 内容 |
|---------|------|------|
| **ケアプラン** | 作成時・更新時 | 個別ケア計画（24時間シート含む） |
| **ケース記録** | 毎日 | 日々のケア内容・状態観察 |
| **バイタル記録** | 毎日（看護） | 体温・血圧・脈拍 |
| **食事摂取量** | 毎食 | 主食・副食の摂取量 |
| **排泄記録** | 都度 | 排泄時刻・状態 |
| **入浴記録** | 入浴時 | 入浴実施日・皮膚状態 |
| **服薬記録** | 都度 | 薬の種類・時間 |
| **事故報告** | 発生時 | 転倒・誤嚥など |

### ⚠️ でも、これは「最低限」

法律は「記録すべき項目」を定めていますが、**「その人らしさ」を記録しろとは言っていません**。

**法律が求めるもの**: 「食事を摂取した」という事実  
**ケアの質を高めるもの**: 「コーヒーを楽しみにしている」という発見

## 記録がケアの質を変える

### 🔄 記録 → 気づき → 改善 → より良いケア

#### ❌ 悪い例：記録が「作業」になっている

```
08:00 朝食 摂取量8割
10:00 排泄介助 実施
12:00 昼食 摂取量9割
```

→ **何も見えない。岡田さんの「人」が見えない。**

---

#### ✅ 良い例：記録が「観察」になっている

```
08:00 【朝食】
和食セット。味噌汁は「熱いのがいいね」と満足そう。
コーヒーを2杯飲まれ、「これで目が覚める」と笑顔。
主食8割、副食9割摂取。

→ 気づき: コーヒーが食事の楽しみになっている。
   朝食時のコーヒーは習慣として継続したい。
```

---

### 💡 記録が変えたケアの実例

#### ケース1: 睡眠リズムの発見

```
【記録からの発見】
- 3/15: 「6:50に目が覚めたけど、まだ眠い」
- 3/16: 「もう少し寝たかったな」
- 3/17: 「7:00は早いね」

【改善アクション】
起床時刻を7:00→7:30に変更

【結果】
- 3/20: 「よく寝たよ、すっきりした」
- 朝食の摂取量が7割→9割に向上
- 午前中の活動性が上がった
```

---

#### ケース2: 好みの発見

```
【記録からの発見】
- 「昔は毎朝コーヒーを飲んでいた」
- 「娘が入れてくれたコーヒーが好きだった」
- 朝食後にコーヒーを飲むと笑顔が増える

【改善アクション】
1. 朝食後に必ずコーヒーを提供
2. 家族面会時にコーヒーを一緒に飲む時間を作る

【結果】
- 朝の目覚めが良くなった
- 家族との会話が増えた
- 「ここに来てよかった」という発言
```

---

#### ケース3: 季節の楽しみ

```
【記録からの発見】
- 中庭の桜を見て「きれいだねぇ」と何度も言われる
- 「来年も見られるかな」という不安
- 桜の話をすると笑顔が増える

【改善アクション】
1. 桜の開花時期に毎日中庭散歩
2. 居室から桜が見える位置に椅子を配置
3. 桜の写真を部屋に飾る

【結果】
- 「来年も一緒に見よう」と前向きな発言
- 午後の気分が安定
- 家族にも桜の写真を共有し、喜ばれた
```

---

### 🎯 記録がないと、このサイクルは回らない

```
記録なし → 気づきなし → 改善なし → 画一的なケア → 満足度低下

記録あり → 気づきあり → 改善あり → 個別ケア → 満足度向上
```

## なぜ「楽に集約できる仕組み」が必要なのか

### 😰 現場の現実：記録が「負担」になっている

#### 典型的な1日のスケジュール（介護職員・日勤）

```
07:00  起床介助（10名分） - 声かけ、着替え、洗面
08:00  朝食介助（10名分） - 食事介助、服薬介助、口腔ケア
09:00  排泄介助・トイレ誘導
10:00  【記録タイム】← やっとパソコンの前に座れる
       でも...
       - 朝の10名分の記録を書く
       - バイタル記録を転記
       - 食事摂取量を記録
       - 排泄記録を記録
       - 気づいたことをメモ
       → 「あれ、岡田さん何て言ってたっけ？」（忘れてる）

11:00  レクリエーション準備
12:00  昼食介助（10名分）
13:00  排泄介助
14:00  【記録タイム】← また記録
       でも午後のケアが待ってる...
       → 「細かいことは後で」（結局書かない）

15:00  おやつ提供
16:00  入浴介助（週3回、1名30分）
17:00  申し送り準備
18:00  夕食介助（10名分）
19:00  【記録タイム】← 疲れてる
       → 「最低限だけ書こう」

20:00  就寝介助
21:00  やっと帰宅準備
       → 残業して記録を書く（サービス残業）
```

---

### 💔 記録が「作業」になる悪循環

```
忙しい → 記録の時間がない → 最低限しか書けない
       → 「気づき」が記録されない
       → ケアが改善しない
       → 画一的なケアが続く
       → 職員も達成感がない
       → モチベーション低下
       → さらに忙しくなる
```

### 🚀 「楽に集約できる」とは？

#### 従来の記録方法の問題点

| 問題 | 詳細 |
|------|------|
| **バラバラ** | 紙の日誌、Excelシート、別システム、メモ帳... |
| **重複入力** | 同じ内容を複数箇所に転記 |
| **検索不可** | 過去の記録を探せない |
| **共有困難** | 夜勤者が日勤者の記録を見られない |
| **分析不可** | 傾向を見つけられない |

---

#### 理想の記録システム

| 機能 | メリット |
|------|---------|
| **一元管理** | すべての記録が1か所に |
| **ワンクリック記録** | 「コーヒー飲んだ」→クリック→記録完了 |
| **音声入力** | 「岡田さん、今日は笑顔が多かった」→記録完了 |
| **自動集約** | 食事・排泄・バイタルが自動グラフ化 |
| **AI提案** | 「最近8:15起床が多いです。起床時刻を変更しますか？」 |
| **リアルタイム共有** | 夜勤者が日中の記録をすぐ確認 |
| **家族共有** | 家族アプリで今日の様子を写真付きで送信 |

---

### 💡 楽になると、何が変わるか？

#### Before: 記録が「負担」

```
職員: 「記録書かなきゃ...（ため息）」
     「細かいこと覚えてない...」
     「最低限だけ書いて終わり」

結果: 画一的なケア、気づきなし、改善なし
```

---

#### After: 記録が「楽しみ」

```
職員: 「あ、岡田さん今日コーヒー2杯飲んでた！記録しよう」
     （スマホでサッと入力、10秒で完了）
     
     「最近8:15起床が多いって、AIが教えてくれた」
     「じゃあ起床時刻変えてみよう」
     
     翌日...
     「岡田さん、よく寝たって笑顔だった！」
     （すぐ記録、みんなで共有）

結果: 個別ケア、気づき多数、改善サイクル、達成感
```

### 🔄 14日間のサイクル

```
Day 1-3: 【Initial - 初期観察期】
- まずは入居者を知る
- 24時間シートの初版を作る
- 「この時間に何をする」を計画

Day 4-7: 【Logging - 記録蓄積期】
- 毎日の記録を蓄積
- 「実際はどうだったか」を観察
- 気づきを付せんでストック

Day 8-11: 【Fitting - 調整期】
- 記録から傾向を分析
- AIが「時間のフィット提案」
- スタッフが「好みのフィット提案」
- 24時間シートを調整

Day 12-14: 【Confirmed - 確定期】
- 調整後のケアを実践
- 本人の反応を確認
- 新しい24時間シートを確定

→ 次の14日間サイクルへ（さらに磨く）
```
